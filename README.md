ТЗ: Функционал корзины
1. Пользователь может добавлять товары в корзину.
2. Пользователь может добавлять не более 5 различных товаров в корзину.
3. Пользователь может добавить в корзину от 1 до 10 единиц одного вида товаров.
4. Суммарное количество всех товаров в корзине не может превышать 20 штук.
5. При попытке добавить более 5 видов товаров, превышающий лимиты, система показывает сообщение: "Лимит видов товаров превышен".
6. При попытке добавить более 10 единиц одного товара, превышающий лимиты, система показывает сообщение: "Лимит видов товаров превышен".
